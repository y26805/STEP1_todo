STEP1_todo
===========

ToDoリスト

製作環境
・言語：HTML, CSS(Bootstrap), JavaScript(jQuery)
・文字コード：HTML, CSS, JavaScriptはUTF-8
・OS：Mac 10.9.5
・ブラウザ：Chrome

基本機能：
・投稿はチェックと内容からなる。
・投稿データはローカルストレージに保存する。
・新しい投稿ほど上に表示される。
・HTMLタグが入力された場合はエスケープする。

追加した機能：
・YES/NO選択。答えによって表示する内容が少し異なる。
・締め切り日を入力。（※時間も入力できるようにしたかったのだが、あいにく時間が足りなかった。）
・残り時間の計算。
・残り時間による警告。残り時間が4日以内のであれば、残り時間は赤字で表示。
・タスクのクリアあるいは削除。
